VIRTUAL PAINTER

An AI-based virtual painter that can be used to draw on screen with RGB colors.
How is it built:
Hand Detection Module : Class containing the methods to detect hands & hand's postion.
Python's cv2 and mediapipe libraries : Provide the modules and methods to get hand landmarks and draw shapes to mark them.

VIRTUAL MOUSE

An AI-based virtual mouse that can be used to navigate on your computer by just using your hand.
How is it built:
Hand Detection Module : Class containing the methods to detect hands & hand's postion.
Python's cv2 and mediapipe libraries : Provide the modules and methods to get hand landmarks and to move the mouse using hand movements.
